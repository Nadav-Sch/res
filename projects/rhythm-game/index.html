Rhythm Game

Role Solo Developer
Tools p5.js, p5.sound
Category Games / Creative Coding
Song Harder Better Faster Stronger — Daft Punk

Press G for the left lane  ·  H for the right lane  ·  Hit notes as they reach the target zone

Overview

A rhythm game built for Creative Coding class using p5.js and p5.sound. Set to Daft Punk's "Harder Better Faster Stronger," the game puts you up against an animated boss enemy. Notes scroll down two lanes and you press G or H to hit them in time with the music — land enough hits to defeat the boss before your health runs out.

How It's Made

The game runs entirely in the browser using p5.js for rendering and p5.sound for audio playback and beat synchronization. Notes are spawned based on a hand-authored beat pattern array that maps to the rhythm of the song. The boss reacts to hits and misses in real time with frame-based sprite swaps — each attack and flinch is a separate pixel-art sprite. The canvas scales responsively to the container width.

Challenges

Syncing note timing precisely to audio without a dedicated game engine was the trickiest part. p5.sound's currentTime() doesn't always start at exactly zero, so the beat pattern had to account for a pre-roll offset. Pixel art sprites also needed careful positioning since p5.js doesn't have a built-in sprite system — each frame is manually drawn and swapped based on game state.
